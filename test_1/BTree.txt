
Склонить с гита в любую папку
	git clone https://github.com/BehaviorTree/BehaviorTree.CPP.git

Установить зависимости
	BehaviorTree.CPP$ sudo apt-get install libzmq3-dev libboost-dev

Построить
	mkdir build ; cd build
	cmake ..
	make
	sudo make install

Для проекта нужно 
	.cpp
	.xml
	CMakeLists.txt
